Project Examples

  • LIRI Node App (coming soon)
    This project, focused on using Node, is a Language Interpetation and Recognition Interface. It is a command line app that takes in specific parameters, and returns data from different APIs.
  • Giftastic
    This is a simple search engine using the API Giphy for data. The user is able to add categories for search, view results, and to stop or start the animation of any gif on the page.
  • Train Scheduler
    This app prompts the user to enter a name, destination, initial arrival time, and frequency for a train. The program takes in that information and adds it to the train schedule, calculating the amount of minutes until the next train. Added trains continue to exist in the schedule even after refreshing the browser.
